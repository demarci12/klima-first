# klima-first

This is an Astro project scaffolded into the current workspace.

## Getting started

1. Install dependencies (already done):
   ```bash
   npm install
   ```

2. Start the dev server:
   ```bash
   npm run dev
   ```

3. Open the dev URL shown in the terminal (usually http://localhost:3000).

---

## Files to know

- `src/pages/index.astro` – the default homepage.
- `astro.config.mjs` – Astro configuration.
- `package.json` – scripts and dependencies.
